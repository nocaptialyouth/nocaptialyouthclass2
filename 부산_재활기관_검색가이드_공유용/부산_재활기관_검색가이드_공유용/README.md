# 부산 재활기관 검색 가이드

`index.html`을 더블클릭하면 별도 설치 없이 사용할 수 있습니다.

- 기관명·구·군·질환·서비스 통합검색
- 재활의학과, 입원재활, 요양병원, 회복기, 산재, 자동차보험, 한방재활 빠른 필터
- 기관별 전화·네이버지도·근거 자료 바로가기
- 원자료에 확인된 홈페이지는 바로 연결, 미확인 기관은 공식 홈페이지 검색 연결
- 상단 `실손24서비스` 바로가기
- 관심기관 저장, 인쇄, 모바일 반응형 화면

> 이 자료는 2026년 7월 공개자료 기반 탐색용 가이드입니다. 방문 전 반드시 전화로 최신 정보를 확인하세요.
